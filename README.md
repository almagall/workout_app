# Workout Planner Web App

An intelligent gym workout planner and dashboard built with Next.js, Supabase, and TypeScript.

## Features

- User authentication (sign up/login)
- Workout plan selection (Hypertrophy or Strength Training)
- Custom workout template creation
- Intelligent workout logging with progressive overload
- Performance tracking and analytics dashboard
- Real-time feedback and suggestions

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables in `.env.local` (already configured)

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Language**: TypeScript
